# PinkBuds Commerce Platform

Production Ready Ecommerce Platform built using

- Next.js
- React
- TypeScript
- Tailwind CSS

## Features

- Product Catalog
- Shopping Cart
- Checkout
- Coupon
- Delivery Options
- Wishlist
- Orders
- Admin Dashboard

## Development

```bash
npm install

npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- Cloudinary
- Razorpay
